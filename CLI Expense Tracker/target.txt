# Concepts You’ll Actually Learn

## Real Python Structure

Instead of one huge file:

```text
expense_tracker/
│
├── main.py
├── storage.py
├── expense.py
├── analytics.py
├── data.json
```

This teaches:

* modular programming
* file separation
* code organization
* scalable thinking

# Required Challenges

## 1. JSON File Handling

Learn:

```python
import json
```

Practice:

* save data
* load data
* update data

This is extremely important.

---

## 2. Error Handling

Use:

```python
try:
except:
```

Handle:

* invalid input
* wrong menu choices
* empty expense lists
* corrupted JSON

Real developers spend huge time handling edge cases.

---

## 3. Clean Code

Avoid:

* giant functions
* repeated code
* random variable names

Practice:

* reusable functions
* clean naming
* readable structure
